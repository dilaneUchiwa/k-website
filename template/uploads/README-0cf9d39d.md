## Description du projet

Site vitrine professionnel multilingue (FR/EN/ES/ZH) avec backoffice d'administration pour **TOCHE & FILS International Trade Co., LTD**, société de commerce international spécialisée dans l'import/export d'engins de chantier et de véhicules lourds entre la Chine et l'Afrique (Cameroun).

### Domaines d'activité couverts
- Vente d'engins & machines (camions, excavateurs, chargeuses, bulldozers…)
- Cargo maritime & aérien
- Paiement de fournisseurs
- Prestation d'accompagnement & achats en Chine

## Stack technique (résumé)

| Couche | Technologie |
|---|---|
| Framework | Next.js 14+ (App Router, TypeScript) |
| Styles | Tailwind CSS |
| Base de données | PostgreSQL (hébergeur au choix du client) |
| ORM | Prisma |
| Auth | NextAuth.js |
| Médias | Cloudinary |
| Emails | Resend + React Email |
| PDF | React PDF |
| i18n | next-intl (FR / EN / ES / ZH) |
| Hébergement | Vercel |
